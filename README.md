# Hello World Frontend Project

This project is a simple React application that displays the phrase "Hello World".

## Instructions to run the project locally:

1. Ensure you have Node.js and npm installed on your machine.
2. Open a terminal and navigate to the project directory `hello-world-frontend`.
3. Run the command `npm install` to install the necessary dependencies.
4. After the installation is complete, run the command `npm start` to start the development server.
5. Open your web browser and go to `http://localhost:3000` to see the "Hello World" message displayed on the page.
